PDA 调用链路

1
登录取 token POST /echain/public/mslJoint.do?action=generateToken4Msl&tenantName=mslscsyxt

username / password -> data.token

2
获取生产批次 POST /echain/thirdParty/mslJoint.do?action=getBatchList

operationTypeCode / page / rp -> content[]

3
获取配方明细 POST /echain/thirdParty/mslJoint.do?action=getFormulaDetail

relationInfoId / batch -> materialList

4
扫码校验 POST /echain/thirdParty/mslJoint.do?action=checkExistCode

traceCode -> exists / receiptId

5
投料提交 POST /echain/thirdParty/mslJoint.do?action=batchSaveMaterialLaunchRecord

launchDetailJson / codeContentsJson -> saved

6
回执确认 POST /echain/thirdParty/mslJoint.do?action=confirmReceipt

receiptId -> handshakeStatus ACKED
